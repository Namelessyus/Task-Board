How TaskBoard Works

A simple, role-based workflow where Admins, Supervisors, and Participants work together to manage projects and tasks.

1

Create or Join a Project

Admins create new projects and receive a unique join code. Users who join with the code automatically become Participants.

2

Manage Tasks Based on Your Role

Supervisors create and assign tasks, while Participants choose tasks to work on. Task status updates automatically as work progresses.

3

Track Progress & Stay Updated

Use the Dashboard to view project progress, deadlines, priorities, and personal tasks. Get reminders and stay organized with real-time updates.

Projects & Tasks: How They Work

TaskBoard uses a clear, role-based system to keep every project organized and every task on track.

Project

A project is the main workspace created by an Admin. Each project comes with a unique join code that members use to enter. Supervisors manage the project details, view members, and create tasks, while Participants can join the project to contribute.

Task

A task is an activity or responsibility inside a project. Supervisors create tasks and set deadlines or priorities, and Participants can choose tasks or be assigned to them. Each task updates automatically as Pending, In Progress, Completed, or Incomplete based on member activity.
